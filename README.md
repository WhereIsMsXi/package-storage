# @where_is_mr_li/storage

> 对 web storage 的增强和封装

## 安装依赖

github:

```bash
git clone https://github.com/WhereIsMsXi/package-storage

cd package-storage

pnpm i
```

### 打包

```bash
pnpm build
```

### 测试

```bash
pnpm test
```

npmjs:

### 安装

```bash

# npm
npm i @where_is_mr_li/storage

# or yarn
yarn add @where_is_mr_li/storage

#or pnpm
pnpm add @where_is_mr_li/storage
```
